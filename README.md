# DIPLOMAT INTERNATIONAL DELIVERY LTD

A comprehensive logistics and shipping management platform with multi-carrier integration (FedEx, UPS, DHL).

## Features

- Multi-carrier shipping integration (FedEx, UPS, DHL)
- Real-time shipment tracking
- Shipment management dashboard
- Customer tracking portal
- Admin operations panel
- Reporting and analytics
- API-first architecture
- Docker support

## Tech Stack

- Backend: Node.js, Express, TypeScript
- Frontend: React, Redux, TailwindCSS
- Database: MongoDB
- APIs: FedEx, UPS, DHL shipping APIs
- Deployment: Docker, Docker Compose

## Quick Start

```bash
git clone https://github.com/officialdick/Diplomat-Internatinal-Delivery.git
cd Diplomat-Internatinal-Delivery
docker-compose up
```

Services:
- Backend API: http://localhost:5000
- Frontend: http://localhost:3000
- MongoDB: localhost:27017

See docs/SETUP.md for detailed setup instructions.
